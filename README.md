## 简介
**小程序 `我们何时见` 前端源码**
> 使用[`uni-app框架`](https://zh.uniapp.dcloud.io/)  
> UI交互,致敬(chao xi)腾讯官方小程序`我们哪里见`

### 后端源码[传送门](https://github.com/twbworld/dating/)

### 运行
1.  `npm install`
2. 下载"微信开发者工具"并登陆
3. 本前端项目用`uni-app`框架, 下载专用的`HBuilderX`编辑器
4. 配置`manifest.json`
5. 运行: 菜单-运行-运行到小程序模拟器-微信开发者工具
